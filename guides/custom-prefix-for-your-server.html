Custom Prefix for your server

SkyAlumny

April 25, 2021 • 5 min read

Mai wants everyone to customize the bot to their own tastes, hence, Akeno allows server administrator to add their own personal prefix.

You can check the prefix as well as the custom prefix you set just by typing prefix wherever Akeno is in, and she will reply what is her prefix and the prefix you set, if you have one.

To set the prefix, just use the command ak!setprefix <desired prefix>. Prefix cannot be more than 5 characters long.

After you set your custom prefix, the default prefix will still be usable. There is no way to remove the default prefix at all. Akeno's developer reserves the right to revert any prefixes that may violate Discord's Term of Service and/or Akeno's Term of Service.

You can change the custom prefix anytime by using the same command again. Note that only users with 'ADMINISTRATOR' permissions may use this command.
