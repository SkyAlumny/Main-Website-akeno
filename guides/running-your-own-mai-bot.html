Running your own Akeno bot

SkyAlumny

April 25, 2021 • 5 min read

To use discord.js (Mai Bot uses Discord.js to interact with the Discord API), you'll need to install Node.js. If you do have Node installed, but have an older version (i.e. anything below 14.0), you should upgrade to the latest version. Akeno requires Node 14.0 or higher.

Get the project Locally

Run the following commands to get the project locally

git clone https://github.com/maisans-maid/Mai.git # or clone your own fork
cd Mai
npm install
npm build

Create a new .env file in the root directory of the project with the following properties

DISCORD_TOKEN=
MONGO_URI=

# Chatbot id and key, if you want to use the chatbot feature
# contact SkyAlumny#6090 for more info
# [OPTIONAL]

chatbot_id=
chatbot_key=
This is also present on .env_example file on the root directory of the project

DISCORD_TOKEN should be the bot token obtained from Discord's developer panel.
MONGO_URI should be your mongo DB URI, if you want to use the MONGO Database. You can use the bot without this too (not recommended, at some point it might crash the bot), but you won't be able to use commands that rely on Mongo DB.
chatbot_id and chatbot_key are completetly optional and it is required only if you want to enable chat bot system of mai

Start The Bot

You can start the bot by typing in npm start in the commandline.
